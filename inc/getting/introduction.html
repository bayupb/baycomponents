Lowersy - Tailwindcss components library

Start by using the open source tailwindcss library which is very easy to use

Introduction

Lowersy is an open source UI component based on the first Tailwind CSS Framework with dark mode features, Figma design, Vanilla JS, Alpine JS Lowersy provides a clean code method with only 1 sentence, for example: "bg-blue" and provides other functions elements such as dropdowns and others
